# Projeto Rocket - Landing Page

Este é um projeto de desenvolvimento web focado na prática de **HTML5** e **CSS3**. O objetivo foi criar uma página de captura (landing page) moderna, aplicando conceitos de seletores, posicionamento e estilização visual.

## 🚀 Sobre o Projeto
O projeto consiste em uma página informativa sobre o lançamento de uma plataforma/evento, com foco em design limpo e tipografia legível.

### O que foi praticado:
* **Estrutura Semântica:** Uso correto de tags HTML5.
* **CSS Pseudosseletores:** Aplicação de estados como `:hover` em botões e links.
* **Estilização de Variáveis:** Organização de cores e fontes.
* **Versionamento:** Uso de Git para controle de alterações.

## 🛠️ Tecnologias Utilizadas
* **HTML5** - Estrutura do site.
* **CSS3** - Estilização e layout.
* **Git/GitHub** - Controle de versão e hospedagem.
